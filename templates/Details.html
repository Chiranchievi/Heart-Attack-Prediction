Know About The Values In Detail
1. Age: You have to Enter the age of the person whom you want to check
2. CP(Chest Pain Types) - Enter values as 1,2,3 or 4:

Type 1: Typical Angina
Type 2: Atypical Angina
Type 3: Non-Anginal Pain
Type 4: Asymptomatic

3. Trestbps: Resting Blood Pressure (in mm Hg on Admission to the hospital)
4. Chol - Serum Cholestrol in mg/dl
5. FBS (Fasting Blood Sugar > 120mg/dl):

Enter value as 1.0 or 0.0
1.0 = True and 0.0 = False
